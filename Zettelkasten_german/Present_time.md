ending regular verbs

| pronouns        | ending | verb       |
| --------------- | ------ | ---------- |
| Ich             | +e     | lern**e**  |
| du              | +st    | lern**st** |
| er/sie/es + ihr | +t     | lern**t**  |
| wir/sie/Sie     | +en    | lern**en** |
|                 |        |            |
|                 |        |            |

- У глаголов с основой на -t, -d, -n часто добавляется -e-: du arbeitest
- После s, -ß, -x, -z в форме du не добавляется дополнительное «s»: du heißt
