ссылочные квалификаторы
[[annotation]] метовов с помощью квалификаторов `&` и `&&`, что бы указать, что выражение слева должно соотвестовать возращаемому типу. а именно **позволяют ограничить ипользование функции только объектами** [lvalue] или [rvalue]. 
```cpp
class Widget {
public:
	void doWork() &;
	void doWork() &&;
};

```
для прочих уточениний методов служат [[attributes]]
